# Preguiceitor: Code generation for lazy people.

<img align="right" height="260" src="https://raw.githubusercontent.com/ronymmoura/preguiceitor/master/sloth.png">


### Install

```shell
npm install -g yo preguiceitor
```

### Using

#### Model

To generate all models, execute:

```shell
yo preguiceitor model
```

#### DAO

To generate a model, you can execute:

```shell
yo preguiceitor dao <name>
```

You can too generate all dao's executing:

```shell
yo preguiceitor dao all
```

#### Service

To generate a service, you can execute:

```shell
yo preguiceitor service <name>
```

You can too generate all service's executing:

```shell
yo preguiceitor service all
```
